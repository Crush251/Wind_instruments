
此次更新只更新了演奏代码，故只用覆盖web文件夹，新增exec文件夹，trsmusic文件夹，sksarm7程序,client.py客户端

1.在笔记本进入到pi文件夹下，执行以下命令：
```bash
scp -r web exec trsmusic linkerhand@192.168.1.xx:/home/linkerhand/pi
scp sksarm7 client.py kill_play_quick.sh linkerhand@192.168.1.xx:/home/linkerhand/pi
```

2. ssh上去目标机器，给执行权限，检查是否完成覆盖，重新启动客户端自启动
```bash
ssh linkerhand@192.168.1.xx
chmod +x pi 
cd pi
ls
./install_autostart.sh
```
查看是否有exec文件夹,有的话就成功

3.执行重启
```bash 
sudo reboot
```

**注意** ：最好在重启的时候在笔记本上打开server.py

更新完毕的程序可以更改空拍时间，如果没有同时演奏，就打开 sksarm7 选取音乐文件，点击第一个空拍，增加或减少。
